Challenge

This lab was about creating pseudocode for certain tasks and learning to think like a programmer.

Problems

Thankfully we didn't have many problems with the assignment. If we had questions about how to implement a feature in HTML, we just looked up how.

Results

Pseudocode: Baking Cookies

  • Preheat oven
  • Make dough
    • Get out tools and ingredients
    • Mix together ingredients as recipe book says
    • Shape dough into balls
    • Put dough on pan
  • Put in oven
    • Start timer
  • Take out of oven

Pseudocode: Hangman Game

  • // Choose a word from a list of words
  • // Start the game
    • // Print underscores for how long the chosen word is
    • // Get input from user
    • // Check if letter is in the word
      • // If no, draw hangman part
      • // If yes, replace an underscore with the letter
    • // Repeat until hangman figure is complete or word has been guessed, then end the game
